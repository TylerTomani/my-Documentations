Array destructuring

 We can access certain index values with destructuring by doing the following 

        let colors = ['blue','red','yellow','green','orange'];
        let [firstColor , secondColor ] = colors;
    
        console.log(firstColor )
        console.log(secondColor)
        /// returns blue
            red

Object destructuring

            let obj1 = { name: "Tony", instruments: ['guitar','banjo']};
        
            let {name, instruments } = obj1
            console.log(name);
            console.log(instruments);
            // reutrns Tony
                ['guitar','banjo']

Alias destructuring

 We can assign the destructured values to a variable with a new name 

            let obj1 = { name: "Tony", instruments: ['guitar','banjo']};

            let {name: birthname, instruments: music} = obj1
            console.log(birthname)
            console.log(music)
            // reutrns Tony
                ['guitar','banjo']

Destructuring Nested Objects

        let zooAnimals = {
            animalID: 32,
            species: "hippo",
            favoriteFood: "watermelon",
            fullName: {
                fname: "Fiona",
                lname: "Applebalm"
            }
        };

to retrieve just the fname nested inside, do the following 

            let { fullName: { fname } } = zooAnimals;
            console.log(fname)

To get both first and last name do the following 

            let { fullName : { fname ,lname} } = zooAnimals;

Parameter destructuring

 This is cool, you can destructure within a functions parameters 

        let dog = { name : "brooklyn", age: 5, color: "brown"};
            
        function sayHi( { name } ){
            console.log("Hi " + name)
        }
    
        sayHi(dog)

 This is great for using the keydown event listner with keyCode 

            el.addEventListner("keydown",({ keyCode }) => {
                console.log(keyCode)
            })

Asigning to a nested object from object

            let object = { animal: { name: "Fiona", species: "Hippo" } };
        
            // here we are specifying that within the animal object we want to assign the
            // *species* variable to the value held by the *species* key
            let {
                animal: { species }
            } = object;
        
            console.log(species); // => 'Hippo'
