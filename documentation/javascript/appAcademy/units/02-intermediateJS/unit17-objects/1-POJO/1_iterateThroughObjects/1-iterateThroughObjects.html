Iterating through objects

 Objects are unordered key and value pairs. Each key is known as a property, and is a string representing a property name We have to iterate through them differently.

  • Iterate through Object keys and values with for...in
  • Use Object.keys and Object.values methods to iterate through Objects

 Example of how to iterate through keys

let obj = {name: "Rose", cats: 2};
for(let key in obj){
 console.log(key);
}
//prints
name
cats

 This is how we access the values within the keys

let obj = {name: "Rose", cats: 2};
for(let key in obj){
 let value = obj[key];
 console.log(value)
}
// prints
Rose
2

 You can NOT use Object.dot notation when iterating. You can only use Object[bracket] notation.

Methods vs Functions

 A method is a function that belongs to an object. Every Method is a function but not every function is a method

  • myFunc is a function
  • myObject.myFunc is a method of the object myObject
  • myObject["myFunc"] is a method of the object myObject

 We can declare the object's methods within the object itself

let dog = {
 name: "Wishbone",
 bark: function(){
  console.log('bark bark!')
 }
}

 Also, we can declare a method using Object.dot or Object[bracket] notation. The key is the method's name, and the value is the function's definition

let dog = {
 name: "Wishbone"
}
// .dot notation
dog.bark = function(){
 console.log("bark bark!")
}
// [bracket] notation
dog["speak"] = function(string){
 console.log(dog.name + " likes " + string)
}
dog.bark();
dog.speak("treats")

Object.keys() method

 The Object.keys() method will return an array containing each key of the object passed in

let employee = {name: "George", age: 32};
console.log(Object.keys(employee))
//prints ['name','age']

Object.values() method

 The Object.values() method will return an array containing all of the values in a object.

let employee = {name: "George", age: 32};
console.log(Object.values(employee))
//prints ['George',32]

Object.entries() method

 Object.entries() will return an array with both key and value pairs.

let animal = {name: "Hank", species: "Giraffe"}
console.log(Object.entries(animal))
// prints [ ['name','Hank'], ['species','Giraffe'] ]
