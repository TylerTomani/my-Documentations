Primitive vs Reference

Primitive

Primitive type variables are inmutable

boolenas,null, undefinedstrings, and numbers are primitive types meaning you cannot change their values in memory

  • booleans
  • null
  • undefined
  • strings
  • numbers

Reference

Reference type variables are mutable

Objects are reference type variables, meaning if you assign a variable to another object and change a property, the original object's property will also change
