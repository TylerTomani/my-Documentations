1-Primitive In Depth

 As of the JavaScript ECMAScript 2015 release, there are now eight different data types in JavaScript.

Primitive

  • 1. Boolean
  • 2. Null
  • 3. undefined
  • 4. Number
  • 5. String
  • 6. Symbol
  • 7. BigInt

Other Reference Type

  • 8. object
  • (An array is a type of object

 Remember,the definition of a method is a function that belongs to an object.

Primitive data types do not have methods that can be performed on them

        const corgi = {
            name: "Ein",
            func: function() {
                console.log("This is a method!");
            }
        };
        
        corgi.func(); // prints "This is a method!"

 We can pass primitive data types into functions but we can NOT declare methods on them 

            // This works because we are calling the Math object's method sqrt
            let num = Math.sqrt(25); // 5
            
            // This will NOT work because the Number primitive has NO METHODS
            let num = 25.sqrt; // SyntaxError: Invalid or unexpected token

The string exception

 A string is a primitive data type, but Javascript wraps strings into object which then allow us to perform methods on them.

 To clarify the String object wraps the String primitive type, and we perform the method toUpperCase() method on the String Object, NOT the primitive strings. JavaScript does this automatically. 

            let str1 = "apple";
            
            str1.toUpperCase(); // returns APPLE
            
            let str2 = str1.toUpperCase();
            
            console.log(str1); //prints apple
            console.log(str2); //prints APPLE

 So in the above example when we call str1.toUpperCase() we are calling that on the String object that wraps around the String primitive type.
