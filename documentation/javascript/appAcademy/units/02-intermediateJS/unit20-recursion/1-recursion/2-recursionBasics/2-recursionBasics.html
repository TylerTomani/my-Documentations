Recusion Basics

 Recusion is the processs of calling a function from within itself.

researchers began using the term "recurse" to refer specifically to functions that are being called from within themselves. Creating a new word by removing a suffix in this way is known as back-formation.

We'll prefer "recurse" when discussing this topic, but you may see "recur"

 Recusion has two cases, case are the expected output for the input. The two cases are known as the base case and recursive case.

  • the base case runs once and ends, also known as the terminating case
  • the recursive case causes the function to call itself

  In the dump() function below the base case is the else if gentlyUnpackFruit()


function gentlyUnpackFruit(contents) {
 console.log("Your " + contents + " have been unpacked!");
}

function dump(crate) {
 if (crate.content_type === "crate") {
  dump(crate.contents);
 } else if (crate.content_type === "fruit") {
  gentlyUnpackFruit(crate.contents);
 }
}

 Notice how the dump function itself is called, if, the contents of the argument passed in is crate. This will execute the remaining code in the crate function that comes after the else if().
