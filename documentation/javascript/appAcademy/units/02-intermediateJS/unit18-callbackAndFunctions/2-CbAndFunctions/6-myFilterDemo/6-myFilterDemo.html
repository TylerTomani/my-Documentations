 fllter() fills a new array with items from another array based on a boolean condition.
This example takes every word from the original array that contains the letter a and puts them into a new array. 

            let food = ['lettuce', 'celery', 'watermellon', 'pineapple'];
            let result = food.filter(function(el){
                return el.includes('a');
            })
            console.log(result)
            // prints Array(2)
            0:"watermellon"
            1:"pineapple"

 You can also write it out this way 

            let result = food.filter(function(el){
            // return el.includes('a')
                if(el.includes('a')){
                    return el
            }
            })
            console.log(result)

This will produce the same result.
