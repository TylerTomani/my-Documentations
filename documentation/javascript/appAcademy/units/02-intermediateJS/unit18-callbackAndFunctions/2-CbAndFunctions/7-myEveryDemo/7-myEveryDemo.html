7-myEvery()

  Just like forEach(),map(), andfilter(), the every method requires a callback as seeen at line 2, but unlike the previous methods, the every() function returns a boolean. 

    1   let food = ['lettuce', 'celery', 'watermellon', 'pineapple'];

    2   let result = food.every(function(el,i,array){
    3       return el.includes('e')
    4   })
    5   console.log(result)
    //prints true
    1   let food = ['lettuce', 'celery', 'watermellon', 'pineapple'];

    2   let result = food.every(function(el,i,array){
    3       return el.includes('x')
    4   })
    5   console.log(result)
    //prints false

 Just like the other methods, we can create the callback and place it into every()'s argument 

    let includesE = function(el){
        return el.includes("e")
    }
    let result = food.every(includesE);
    console.log(result)

 To clarify, the return boolean value will be true, ONLY, if every array item meets its criterion
