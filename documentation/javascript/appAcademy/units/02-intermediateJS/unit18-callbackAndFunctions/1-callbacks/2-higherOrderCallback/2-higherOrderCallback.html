2-Higher order vs callback functions

 Just like strings,booleans,and numbers, functions are first class objects meaning they can be passed in and returned from functions

 If we assign the value of a variable to a function, whatever that functions returns becomes that value

let foo = function(){
 let val = "Tacos"
 return val
}

let result = foo();
console.log(result)
//prints Tacos

 If weconsole.logfoo, Thefoo function itself will be printed. But, If the return value of foo is a function, and if we invoke the foo function inconsole.log, the bar function being returned will be printed

let foo = function(){
 let bar = function(){
  console.log("interesting...")
 }
 return bar
}
console.log(foo)
// prints [Function: foo ]
let foo = function(){
 let bar = function(){
  console.log("interesting...")
}
 return bar
}
console.log(foo)
//prints [Function: bar ]
