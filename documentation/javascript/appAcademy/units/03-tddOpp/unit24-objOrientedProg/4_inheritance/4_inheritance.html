4_inheritance

 All Javascript is derived from the root being an object within itself. The starting window is an object. So when you declare a class with no explicit parent class, then JavaScript will make it a child of Object.

class MyClass {}
// Is the same as
Class MyClass extends Object {}

 We use extends to inherit all the properties from a parent class. It inherits the methods of the parent class.

 For example, if a class Animal is defined with constructor and speak instance methods and a pet static method, then any class, like a Dog, that extends it will be able to use those methods.

class Animal {
 constructor(name,sound){
  this.name = name;
  this.sound = sound;
 }

 speak(){
  console.log(`${this.name} says "${this.sound}"`)
}
 static pet(animal){
  console.log(`You attempted to pet ${animal.name}`);
 }
}
class Dog extends Animal{};
const fluffy = new Dog("fluffy","woof");
fluffy.speak();
Dog.pet(fluffy);
//prints
fluffy says "woof"
You attempted to pet fluffy

 If you want to define a new constructor method in the child class but still use the code in the constructor method of the parent class, you can use the super

 From the example above, if we make a Dog class from the parent class Animal and want to make the default sound value for all instances of Dog to "woof", use super inside of the Dog constructor and insert "Woof" as the second parameter 

class Dog extends Animal{
    constructor(name){
        super(name,"woof");
    }
};
 const skip = new Dog("skip");
 skip.speak();
 Dog.pet(skip);
// prints
skip says "woof"
You attempted to pet skip

Get name of the object

 To get the name of the object use, constructor.name 

class Fruit {
    constructor(color,taste){
        this.color = color;
        this.taste = taste;
    }
    eating(){
        console.log(`The ${this.constructor.name} taste ${this.taste}`);
    }
}
class Apple extends Fruit{};
const apple = new Apple("red","bitter")
