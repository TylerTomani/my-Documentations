6_Use Parent Methods with super

 Inside a child class we can build on top of a parent's method.

class ExampleParent{
 print(arg){
  console.log(arg);
 }
}

class ExampleChild extends ExampleParent {
 print(args){
  args.forEach(arg => super.print(arg))
 }
}
const numArray = [1,2,3,4];
const newChild = new ExampleChild
newChild.print(numArray)
// prints
1
2
3
4
