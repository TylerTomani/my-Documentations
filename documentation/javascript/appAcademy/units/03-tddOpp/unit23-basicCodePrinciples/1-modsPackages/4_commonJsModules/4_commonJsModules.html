4-Common Js Modules

 In Node.js, each JavaScript file defines a module. Each module has access to a module object which has the property module.exports which is used to export modules

Exporting multiple items from a module

 Create some functions then define them in the module.exports object

      calculator.js
let add = function(num1, num2){
 return num1 + num2;
}

let negate = function(n){
 return n * -1;
}

module.exports = {
 add,
 negate
};
      calculator.js
let add = function(num1, num2){
 return num1 + num2;
}

let negate = function(n){
 return n * -1;
}

module.exports = {
 add: add,
 negate: negate
}

The above code on the left is the short hand way to the code on the right. Or you can use the . notation 

        module.exports.add = add
        module.exports.negate = negate

Importing using require

To use the above module in another, Destructure what you want from the exported module.

const {negate} = require("./calculator");

let neg = negate(4);
console.log(neg)

 When a module needs something from another module, it is said to be dependent on that module.

 You to to navigate to the correct directory when importing. If you are trying to import the students.js module into the physics.js module do this. Notice you do NOT need .js with require

const students = require("../people/students");
