2. Array split() & join() methods

split()

Turns a string into an array. Inside the ( ) is how the string is divided up. split("") will seperate each letter, split(" ") will seperate the string by each space

let str = "follow the yellow brick road";
let newArr = str.split("")
[
'f', 'o', 'l', 'l', 'o', 'w',
' ', 't', 'h', 'e', ' ', 'y',
'e', 'l', 'l', 'o', 'w', ' ',
'b', 'r', 'i', 'c', 'k', ' ',
'r', 'o', 'a', 'd'
]
let str = "follow the yellow brick road";
let newArr = str.split(" ")
[ 'follow', 'the', 'yellow', 'brick', 'road' ]

The original string is not mutated

join()

join() will turn an array into a string, and whatever is placed within the () will be put between each array element. 

let strArray = ["run","around","the","block"]
let strSpace = strArray.join(" ")
let strDash = strArray.join("-")
let strDollar = strArray.join("$")
console.log(strSpace) // run around the block
console.log(strDash) // run-around-the-block
console.log(strDollar) // run$around$the$block

Use both join and split to replace

This is very useful, combine the two to replace a word within a string

let str = "This is how I roll";
let newStr = str.split("I").join("We");
console.log(newStr)
// This is how We roll
