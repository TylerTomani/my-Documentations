1. Array splice() method

 splice() mutates the array and does not return a new array.
splice() is poweful because it can remove & insert elements into an array

Using splice to remove

The arguments for removal are:

splice(target index, how many elemnents to remove) 
let colors = ["red","yellow","blue","green","orange","brown","grey"];
let returnColors = colors.splice(2,3); 
// [ 'blue', 'green', 'orange' ]

Using splice to add

The arguments for inserting are:

splice(insertion index, 0, ...elements to insert) 
colors.splice(3,0,"forest green","purple")
console.log(colors)
[ 'red', 'yellow', 'brown', 'Forest Green', 'Purple', 'grey' ]

Inserting and removing

 To add and remove, the second argument should be how many you wish to remove from the insertion point 

colors.splice(0,3,"tomato","goldenrod","royalblue")
console.log(colors)
[
'tomato',
'goldenrod',
'royalblue',
'green',
'orange',
'brown',
'grey'
]
