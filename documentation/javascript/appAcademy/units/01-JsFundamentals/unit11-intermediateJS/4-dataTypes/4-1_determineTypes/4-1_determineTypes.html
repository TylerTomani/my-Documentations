1. Determining Types

 To find what a variable's type is use the typeof operator

let str = 'This is a string';
console.log(typeof str)// string

let num = 8.4;
console.log(typeof num ) // number

let sum = function(num1,num2){return num1 + num2};
console.log(typeof sum ) // function

The value returned from the typeof operator is a string. So to to check if something is of a certain data type do the the following,

if(typeof num === 'number'){
console.log("This is a number")
} else console.log("This is not a number")
