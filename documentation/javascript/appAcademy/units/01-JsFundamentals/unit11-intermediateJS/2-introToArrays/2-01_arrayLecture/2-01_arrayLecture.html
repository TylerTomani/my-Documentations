2-01-Array Lecture

We can access a value in an array directly by providing an index

console.log(["a","b","c"][1]); // "b"
console.log(["a","b","c"][2]); // "c"

indexing an element which doesn't exist will produce "undefined

console.log(["a","b","c"][3]); // "undefined"

We can aquire the index of an element with indexOf()

console.log(["1","5","cat","dog"].indexOf(5); // 1
console.log(["1","5","cat","dog"].indexOf("cat")); // 2

If you try to index an element that does not exist, it will return -1

console.log(["1","5","cat","dog"].indexOf(8); // -1

Concatenation

If we use the + operator with arrays it will combine them but also combine index[0] of both arrays into one element

console.log([1,2,3] + [4,5,6])  // [1,2,34,5,6]

Use the concat() method instead of +

console.log([ 1, 2, 3].concat([4,5,6]))  // [1,2,3,4,5,6]
