3-01-Function Expressions

Functions are first class objects which means they can be stored into variables.

There are two types of way to declare a function.

Function Declaration Syntax

function nameOfFunction(ar1,arg2){}

Function Expression Syntax

let nameOfFunction = function(arg1,arg2){}

When functions are placed into variables, the variable contains the function. If you console.log the variable it will print the function. If you call the variable, it will execute that function 

let sum = function(arg1,arg2){
    return arg1 + arg2;
}
let newNum = sum;
console.log(newNum) // [Function: sum]
console.log(newNum(2,5)) // 7
