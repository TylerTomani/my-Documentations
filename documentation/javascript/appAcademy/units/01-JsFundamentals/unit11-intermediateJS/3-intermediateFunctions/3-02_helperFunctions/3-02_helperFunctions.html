Decomposition

// 1. write a yellString() function
2. It takes an array of strings as an argument
3. returns array where every string is yelled

First white a function that performs the action on a single element

function yellStr(str){
return str.toUpperCase();
}
let newStr = yellStr("not yelling");
console.log(newStr) // NOT YELLING

Decomposition is recognizing that the function yellStr() is the sub problem of yellStrings().

yellStr() is the helper function for yellStrings() 

function yellStrings(arr){
    let newArr = [];
    for(let i = 0; i < arr.length; i++){
        newArr.push(yellStr(arr[i]));
    }
    return newArr;
}
const nums = ["one","two","three"];
console.log(yellStrings(nums));
// [ 'ONE', 'TWO', 'THREE' ]

Use helper function to find the laligat Sum

 A number's laligat sum is the the sum of all the prime numbers less than or equal to that number.

the laligat sum of 10 is: 2 + 3 + 5 + 7 = 17
2 + 3 + 5 + 7 + 11 = 28 

// 1. create a function that checks if number is prime
function isPrime(num){
    if(num < 2){
        return false
    }
    for(let i = 2; i < num; i++){
        if(num % i === 0){
            return false;
        }
    }
    return true;
}
console.log(isPrime(5)) // true
console.log(isPrime(4)) // false

// 2. Use the function within a for() loop in an if() statement
as the boolean condition. If the index meets the condition, add it to the total sum
            
function laligatSum(num){
    let sum = 0;
    for(let i = 0; i < num; i++){
        if(isPrime(i)){
            sum += i;
        }
    }
    return sum;
}
console.log(laligatSum(10)) // 17
console.log(laligatSum(12)) // 28

Abstraction

.flex

Abstraction is the process of hiding details. The human brain is relatively bad at reasoning through complexity. As you will see, computer programs become pretty complex rather quickly and it is hard for the human brain to keep up. The job of a programmer is to manage the complexity by keeping the program modular and hiding details. The act of hiding details allows our minds to focus on other aspects of the program that need attention.

When you read the decomposed code for laligatSum, you don't have to have to think much about what goes on when we call the isPrime function. If isPrime is written correctly then we can pretty much tell exactly what it does just from the function name. We use abstraction all the time when we declare variables and define functions.
