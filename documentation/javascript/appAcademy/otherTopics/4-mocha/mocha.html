mocha

 The mocha installation instructions are located on the same page as node js. It is at the bottom and here is the link github nodeJS

 Below is a summarized version of the gihub setup page

 Install it with the npm, Do NOT confuse it with nvm. nvm is for verison control while npm is for packages

 Install mocha by typing:

npm install -g mocha

 In this case it means global. By default, npm will install files into your project directory. We want mocha to be installed for all projects, so we specify -g.

You can verify mocha works by typing

mocha --version
