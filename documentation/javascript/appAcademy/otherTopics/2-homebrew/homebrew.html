homebrew

 The homebrew installation is covered in Unit 3 - Enviroment Setup click the following link to go to the gihub setup page, it will go directly to the homebrew section homebrew

 Summarized Instructions on the github page are listed below

Go to the homebrew website at http://brew.sh and follow the instructions on the main page.

 You will see instructions to run a command like this, so open a Terminal and run this:

/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"

 After some output it should print out that homebrew is installed successfully.

 You can always check homebrew for possible problems by typing this:

brew doctor

*Note, warnings generated can usually be handled by running the following command

brew upgrade
